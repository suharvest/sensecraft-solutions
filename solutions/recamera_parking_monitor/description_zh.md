## 这个方案能帮你做什么

管停车场不一定要给每个车位装地磁。这个方案用一台 reCamera 就能当停车位监控——对着一排车位，实时告诉你哪个空着、哪个有车。

## 核心价值

| 好处 | 具体说明 |
|------|---------|
| 一台摄像头管多个车位 | 一台 reCamera 同时监控最多 3 个相邻车位 |
| 实时状态更新 | 绿色 = 空闲，红色 = 占用，一目了然 |
| 防误判 | 内置稳定算法，不会因为路过行人、阴影变化而误报 |
| 无需服务器 | 全部在 reCamera 上运行——不用联网，不用额外设备 |

## 适用场景

| 场景 | 怎么用 |
|------|--------|
| 小型停车场 | 把 reCamera 对准一排车位，在 Dashboard 上看哪个空着 |
| 车库入口 | 给进来的车主显示还有几个空位 |
| 专用车位 | 监控 VIP 或无障碍车位，被占用时及时发现 |

## 使用须知

### 核心硬件设备

| 设备 | 说明 | 必需 |
|------|------|------|
| reCamera | 任意型号：2002w（WiFi）、Gimbal、HQ PoE | ✓ 必选 |

### 网络要求

- reCamera 通过 USB 连接（默认 IP：192.168.42.1）或 WiFi/有线网络
- 在同一网络的任何设备上都可以访问监控 Dashboard

### 使用限制

- 每台摄像头最多监控 **3 个相邻车位**
- 需要将 reCamera 以略微倾斜的角度正对车位安装（非俯视）
- 光线充足的环境效果最佳
