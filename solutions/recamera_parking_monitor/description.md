## What This Solution Does

Managing parking lots shouldn't require expensive sensors on every spot. This solution turns a single reCamera into a smart parking monitor — it watches a row of parking spaces, tells you which ones are free and which are taken, all in real time.

## Key Benefits

| Benefit | Details |
|---------|---------|
| One camera covers multiple spots | A single reCamera monitors up to 3 adjacent parking slots simultaneously |
| Real-time status updates | See free/occupied status with color-coded indicators — green means free, red means taken |
| Smart false-alarm filtering | Built-in stabilization ignores shadows, pedestrians walking by, and brief obstructions |
| No server required | Everything runs on the reCamera itself — no cloud, no extra hardware |

## Use Cases

| Scenario | How It Works |
|----------|-------------|
| Small parking lot | Mount reCamera facing a row of spots, see which are available on the dashboard |
| Garage entrance | Show incoming drivers how many spots are free before they enter |
| Reserved parking | Monitor VIP or accessible spots and get alerts when they're misused |

## What You Need

### Hardware

| Device | Details | Required |
|--------|---------|----------|
| reCamera | Any variant: 2002w (WiFi), Gimbal, or HQ PoE | Required |

### Network

- reCamera connected via USB (default IP: 192.168.42.1) or WiFi/Ethernet
- Access the monitoring dashboard from any device on the same network

### Good to Know

- Optimized for **3 adjacent parking slots** per camera
- Mount reCamera at a slight angle facing the parking spots (front-facing view, not bird's-eye)
- Works best in daylight and well-lit environments
