## 这个方案能帮你做什么

仓库里的叉车去哪了？员工有没有按时到岗？贵重设备是不是被挪走了？传统方案要么布线麻烦、要么太贵。这套方案用蓝牙定位标签 + 远距离无线传输，让你在电脑或手机上随时查看人员和物品的位置。

## 核心价值

| 好处 | 具体说明 |
|------|---------|
| 找东西快 | 打开网页输入设备名，几秒钟就知道在哪个区域 |
| 免布线 | 定位标签贴墙上就行，电池能用 2 年 |
| 覆盖广 | 一个接收器覆盖 2 公里，整个园区都能管 |
| 自动记录 | 进出区域自动打卡，不用手动签到 |
| 一键求助 | 追踪器带 SOS 按钮，紧急情况长按报警 |

| 场景 | 怎么用 |

| 设备防盗 | 贵重仪器贴上追踪器，被移动时自动报警 |

## 使用须知

### 定位精度

- 精确定位：1-3 米，需每个区域放置 3 个以上定位标签，可查看移动轨迹
- 区域定位：每个房间放 1 个标签，只显示"在哪个房间"

### 安装建议

- 定位标签安装高度 2.5-3 米
- 标签间距 10-15 米
- 金属货架多的地方需多放几个标签

### 电池寿命

定位标签电池寿命 2 年以上，一般可达 3 年

### 节点容量

同一范围内可连接 100-200 个追踪器（取决于 LoRaWAN 网关）

### 输出接口

开源，可对接第三方 LoRaWAN 服务（ChirpStack、TTN 等）
