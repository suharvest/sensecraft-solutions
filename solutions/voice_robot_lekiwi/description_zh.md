## 这套方案能帮你做什么

这套方案将你的 LeKiwi 机器人变成一个能听懂语音指令的伙伴。说出"前进"、"左转"等自然语言指令——机器人会理解你的话，用语音回复确认，然后执行动作。不需要遥控器、不需要手机 App、不需要键盘。你只需要说话。

所有功能都在树莓派上运行——它听到你的声音、理解你的意图、决定做什么，并通过音箱回应你。XIAO ESP32S3 负责控制电机和驱动轮子。部署流程会引导你依次完成固件烧录、舵机 ID 配置和语音容器部署。

## 核心价值

- **解放双手**——说"Hey Jarvis"然后下达指令，机器人听到、理解、执行
- **自然语言**——不用记按键，想怎么说就怎么说——"往前走"、"向左平移"、"一直转"
- **语音回复**——机器人会说话，通过音箱播报确认和状态
- **全向移动**——Kiwi-drive 底盘让机器人能灵活地前进、后退、横移、旋转

## 你需要准备

### 硬件

| 部件 | 用途 |
|------|------|
| LeKiwi 套件 | 三轮 Kiwi-drive 底盘，含 3 个 STS3215 智能舵机 |
| XIAO ESP32S3 | 电机控制器——接收树莓派发来的串口指令 |
| 树莓派 5 | 语音 AI 大脑——运行完整的语音流水线 |
| reSpeaker Flex XVF3800 | 4 麦克风阵列，实现远场语音采集 |
| 音箱 | 音频输出，播放机器人的语音回复 |
| USB 线 | 树莓派 ↔ XIAO、树莓派 ↔ reSpeaker、树莓派 ↔ 电源 |

### 软件和账号

- 树莓派上安装 **Docker**
- **Groq API 密钥**（免费额度——在 [console.groq.com](https://console.groq.com/) 注册）

## 使用须知

- **需要手动组装硬件**——先完成 LeKiwi 底盘的物理组装；舵机 ID 会在步骤 2、3 中通过 XIAO 配置
- **Groq 免费额度够用**——免费 API 配额完全满足日常使用
- **需要联网**——语音识别和 LLM 运行在 Groq 云端，树莓派需要保持网络连接
- **需要音箱**——机器人会语音回复，没有音箱就体验不到完整效果
- **舵机需要外接电源**——STS3215 舵机需要外部供电（不能仅靠 USB 供电）
