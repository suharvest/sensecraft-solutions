## 这个方案能帮你做什么

这个方案可以通过 SSH 一键把 Depth Anything V3 部署到 NVIDIA Jetson，使用预构建 Docker 镜像，用户无需手动输入终端命令。

## 核心价值

| 价值 | 说明 |
|------|------|
| 一键部署 | 在前端点击部署即可完成流程，无需手输命令 |
| Jetson 适配 | 默认包含 GPU 运行时和 Jetson 常用容器参数 |
| 快速落地 | 复用预构建镜像，降低环境准备复杂度 |

## 适用场景

| 场景 | 收益 |
|------|------|
| 机器人原型开发 | 快速准备边缘端深度估计运行环境 |
| 视觉演示项目 | 快速搭建基于摄像头的深度演示 |
| 内部验证测试 | 统一部署流程，便于重复测试 |

## 使用须知

- 目标设备需为支持版本的 Jetson Linux。
- 目标设备需具备 Docker 和 NVIDIA runtime。
- USB 摄像头为可选项，是否使用取决于你的后续业务流程。
